## ğŸ” Research Focus

This website is built around a short, anonymous survey designed to measure **real user behavior and awareness**, not just opinions.

Key research questions include:
- Do users understand what happens to the data they input into AI tools?
- Are privacy and security considerations actively evaluated, or mostly ignored?
- How much trust do users place in AI-generated advice?
- Where do users feel uncertain or under-informed when using AI?

Survey results are presented visually to make trends easy to understand for non-technical audiences.

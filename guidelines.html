Safe & Ethical AI Use

Practical rules to reduce privacy/security risk when using AI tools.

What you should NOT paste into AI tools

  • Passwords, API keys, authentication tokens
  • Social Security numbers, full addresses, banking info
  • Private school/work documents or confidential code
  • Anything you wouldn’t post publicly

Better habits

  • Use placeholders (e.g., “NAME”, “COMPANY”) instead of real identifiers
  • Summarize data instead of pasting the raw file
  • Check privacy settings (history, training, data controls)
  • Verify outputs — don’t blindly trust sensitive advice
